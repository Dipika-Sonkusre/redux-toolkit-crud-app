# React + TypeScript + Vite

A compact React app demonstrating CRUD operations using Redux Toolkit and a feature-rich data table with pagination, client-side searching, and column sorting. Perfect as a starter template or learning reference to integrate state management with table UX.

Key features: <br>
Create / Read / Update / Delete (CRUD) powered by Redux Toolkit slices and async thunks <br><br>

Table component with:<br>
Pagination (page size, page navigation)<br>
Instant searching/filtering<br>
Multi-column sorting (asc/desc)<br>
Clean separation of concerns: UI components, Redux logic, and services<br>
TypeScript-ready (optional) and easy to extend<br><br>

Tech stack:<br>
React<br>
Redux Toolkit (slices, RTK Query or thunks)<br>
Optional: TypeScript, React Router, Material-UI or Tailwind for UI<br><br>

Quick start: <br>
git clone <repo-url><br>
npm install or yarn<br>
npm start or yarn start<br>
